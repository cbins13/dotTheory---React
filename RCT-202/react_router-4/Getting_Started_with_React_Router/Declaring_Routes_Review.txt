1. Which component is one of the core components of React Router and keeps your UI in sync with the URL?
- <BrowserRouter>

2. Which React Router component is responsible for rendering UI (or other components)?
- <Route>

3. React Router uses JSX syntax to declare routes.
- True. React Router is a set of components, and the declarative syntax of JSX makes it easier to visualize how routes are structured.

4. Which <Route> prop renders a component only when the path matches the URL exactly?
- exact 

5. Which <Route> prop accepts an inline function that gets called when the URL and path match?
- render

6. Given this code: <Route exact path="/signup" component={SignupForm} /> Which of the following is true about Route?
- It renders the SigupForm component only when the URL path is exactly '/signup'.
