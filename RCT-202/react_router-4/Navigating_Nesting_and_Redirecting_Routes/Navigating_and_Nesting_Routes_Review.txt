2. Which component lets you add styling to a link when it matches the current URL.
- <NavLink>

3. Given this route:
<Route path='/products' render={() => <Redirect to='/products/books' />} />
Which of the following is true?
- The route renders a redirect to '/products/books' when the URL path is '/products'.

4. Which React Router component allows users to navigate between routes?
- <Link>

5. The _____ prop assigns a custom CSS class to NavLink when it's active.
- activeClassName
