1. Which of the following best describes properties (or props) in React?
- Props pass data from a parent component down to child components.

2. A component can change the value of the props given to it.
- False. Props are "read only" (or immutable). A component can only read the props given to it, never change them.

3. Which property should you provide to components generated by iterating over an array?
- key

4. Pass the score prop the number value 100.
- {100}

5. Enable the use of props inside the Header component. Then, add the code that displays the value of a prop named user.
- props, props.user

6. How should you write an iteration method like map() in JSX?
- It's a JavaScript expression, so it needs to be placed inside curly braces.
